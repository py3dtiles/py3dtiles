py3dtiles

Py3dtiles is a python library and command-line tool designed for generating and managing 3D Tiles datasets, the OGC standard for streaming massive 3D geospatial content.

Py3dtiles aims at being a swiss knife for converting and editing tilesets. It can read and create tilesets and tilesets content. For file conversion, it currently supports various pointclouds format (las, xyz, ply...), wkb and ifc files.
It can be extended to add your own pointcloud reader or geometry tiler. Lib usage allows it to be integrated into a larger application or into data pipelines.
